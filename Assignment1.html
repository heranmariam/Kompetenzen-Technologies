HTML (Hypertext Markup Language)

Hyper Text

It is a text that can produce documents(webpages), that can contain references to other documents.

Markup Language

When a language syntax is in the form of a tag, not in the form of a statement.


HTML stands for HyperText Markup Language.
It is used to design web pages using a markup language.
HTML is a combination of Hypertext and Markup language.
Hypertext defines the link between web pages.

HTML was created by Tim Berners-Lee.
